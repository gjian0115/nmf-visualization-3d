3D NMF visualization
===
geometric visualization of Nonnegative Matrix Factorization (NMF)

## Requirements
- pyqtgraph (`pip install git+https://github.com/pyqtgraph/pyqtgraph`)
- pytorch
- numpy
- scipy

## Usage
```
$ python main.py
```
